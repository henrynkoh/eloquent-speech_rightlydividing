## How to Share Localhost with Partners

If you want partners to access localhost:3000, you have a few options:

**Option 1: Use ngrok (Recommended)**
```bash
# Install ngrok: https://ngrok.com/
ngrok http 3000
# Share the ngrok URL (e.g., https://abc123.ngrok.io)
```

**Option 2: Deploy to Vercel (Easiest)**
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
cd eloquent-speech
vercel

# Share the Vercel URL
```

**Option 3: Share Screen Recording**
- Record your screen showing the app
- Upload to YouTube (unlisted) or Loom
- Share the link
